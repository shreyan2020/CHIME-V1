INTRODUCTION

In this study, we are interested in characterising which objects in images cause a certain outcome in a Computer Vision model. For this purpose, you will see the original image (on the left) and a processed version of it (on the right), highlighting what the model thinks is important when making a decision. The warmer the colour, the more important that patch is. That is, blue means low importance, and red means high importance.

You are invited to tag the objects, and related properties (like shape and colour), that you think the model is looking at. You will be doing this for 5 different images.

GENERAL INSTRUCTIONS:

  • The background of an image can also be considered as an object to be tagged;
  • The suggestions in each field are provided for guidance, you can enter values outside of those provided
  • You must complete annotate all 5 images to complete the task;
  • Press "Next" when you are done listing down all the objects for an image to move to the next image;
  • At the end, you will be automatically redirected back to Prolific;
  • An image might display multiple objects, use the "Add More" button on the right to add more than one object;
  • You can also remove any concept at any time by clicking the "Remove" button beside it.

DEMO: Below is a video that showcases how to use the application

CASES: Following cases can occur

CASE 1: The entirety (or majority) of an object is highlighted

  • Object name: the name of the highlighted object;
  • Colour: the colour of the object as in the original image;
  • Shape: the shape of the object as in the original image;

CASE 2: A small portion of the object is higlighted

  • Tick "Part of an Object?" - Which Part?: here you can specify which part of that object has been highlighted. For this, try to be as specific as you can. For example, "leg" for the leg of chair;
  • Main Object Name: the name of the object;
  • Colour of the part: the colour of the highlighted part as in the original image;
  • Shape of the part: the shape of the highlighted part as in the original image;

NOTE: If you are not sure about whether case 1 or case 2 applies for your task, you can add both
